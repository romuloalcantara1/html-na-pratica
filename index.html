Inspetor de Elementos 🔎

O inspetor de elementos é a ferramenta do desenvolvedor, que fica embutida no navegador. Para abrir a ferramenta basta apertar F12 no navegador Chrome, liberando assim o console. O mesmo tem vários tipos de recursos para entender como uma página foi criada e seus elementos. Por exemplo, basta clicar no icone de seleção e posicionar no elemento pra ver como ele foi construído. Também é mostrado o código HTML do elemento e o CSS, que vai formatar e deixar bonito o elemento. Dá até para alterar as cores e as características dos elementos do site para estudar a página.

Estrutura Básica do HTML 🧱

Nesta aula foi definida uma pasta onde vai ficar o projeto, assim basta ir em arquivo, abrir pasta e selecionar a pasta que quiser trabalhar no Visual Studio Code. Apareceu toda a estrutura de arquivo do lado esquerdo. Dentro desta pasta criamos uma página index.html que é uma convenção para página principal. O HTML consiste em três partes: Primeiro a tag <html> que é aberta dentro de um sinal de maior e sinal de menor e tem o fechamento da tag com a barra. Segundo tem a tag <head> que é o cabeçalho e terceiro a tag <body> que é o corpo. Tudo que tiver no corpo é o que de vai aparecer para o usuário. No head tem a tag <title> que é o título da página. Para usar o Live Server que instalamos é só clicar com o botão direito no arquivo e clicar em Open with Live Server.Assim, toda vez que alterar e salvar, o arquivo atualiza automaticamente a página em construção no navegador.

Falando sobre Tags 🔖

Na aula foi falado sobre o funcionamento das tags e que existem várias tags. A tag <I> significa Itálico, para colocar em negrito se utiliza a tag <strong>. Dá pra colocar ambas as tags juntas, basta abrir e fecha-lás corretamente pois o seu comportamento se dá pela abertura e fechamento com o conteúdo dentro das mesmas. Algumas tags não precisam de fechamento como a <input>, pois ela sozinha já cria um campo com o fechamento e barra no final. Outra tag que auto se fecha é a <img>.

Atributos das Tags 🔖

Atributos são prorpriedades que a tag pode ter, alguns sao mais globais e outros sao mais específicos. A tag <strong> pode ter o atributo "id" que identifica esse elemento. Por exemplo o CSS pode pegar esse elemento por meio desse atributo e alterar esse elemento. Outra tag chamada style, voltada para o CSS, coloca um comando diretamente no elemento como o color. Também voltada para o CSS, o atributo "class", é uma forma de padronizar tipos em uma página. Já atributos específicos são por exemplo o type que é um atributo da tag <input>. Outro atributo específico é o tipo "color" que gera um campo de cor na página HTML.Basicamente se usam o "text" e o "number" para a tag <input>. Outro atributo específico é o <img> que precisa do atributo "src" para ser colocado o endereço de uma imagem para ela aparecer na página e o atriburo "width" que vai controlar a largura da imagem.

Textos 📄

Nessa aula falamos sobre a tipografia html. A tag <h1> significa header, ou seja, título da página. Quando desenvolvemos texto existe nível e subníveis do <H1> até o <H6>. Dentro do <H1> temos o subtitiulo que é o <H2>. O corpo do texto é definido pelo <p> de parágrafo, que é uma quebra de linha para um texto novo. O espaçamento entre os títulos e parágrafo no navegador podem ser alerado via CSS. Através da ferramenta do desenvolvedor do navegador é possível ver como a estrutura do texto está se comportando, como as margens externas do texto. Os textos na página da web se comportam com responsividade. A tag <bloquote> faz um citação de outro autor ou observação, o navegador dá um espaçamento e uma margem, desprendendo do texto. Podemos colocar outras tags no texto como a <strong> para deixar em negrito, a tag <u> para deixar em underline e colocar a tag <i> como itálico. A tag <mark> deixa o texto amarelo como marca texto.Se quiser colocar um texto sobescrito pode-se usar a tag <sup>.

Listas Ordenadas e Não Ordenadas 📋

Existem dois tipos basicamente de listas, as listas ordenadas e as não ordenadas. A tag <ol> é a lista ordenada, essa tag precisa da tag filha, que está dentro de outra, chamada de pai. Para implementar o item na lista precisa colocar o item na tag <li>. Nesse tipo de lista tem-se uma ordem como 1..2..3. Dentro de uma <li> você pode colocar outra lista. Nas listas não ordenadas utilizamos a tag <ul>, em que os itens são listados por marcadores. É muito utilizado para menus porque via CSS dá para alinhar os elementos.

Links 🔗

Todo site quando clicamos em links, vamos para outras paginas e assim navegamos na internet. Vamos linkando uma página na outra. No HTML usamos a tag a<a> que é uma âncora que vai para algum lugar. Para colocar um site colocamos a tag <a href "site"></a>, essa tag abre e fecha. Entre as tags colocamos um texto que será clicado. Para o link abrir em uma nova aba pode-se usar o atributo target="_blank". Atributo "title" faz com que apareça uma mensagem quando posiciona o mouse sobre o link. Para navegar entre páginas no mesmo site basta digitar o link da página no index por meio das tags.
